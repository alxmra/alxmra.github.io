Detailed Description

Shared memory management.

(implementation file)

Operations defined on shared memory:

  • creation of a new block
  • connection to a previously created block
  • destruction of a previously created block
  • mapping of the block previously created on the process address space
  • unmapping of the block off the process address space.
Author
António Rui Borges - October 1995
